# Golden Drums Slot

Static HTML slot game prototype.

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server.

```bash
npx serve .
```

## GitHub Pages Deployment

This project is ready for GitHub Pages because it is a static site with `index.html` at the repository root.

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings > Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select the `main` branch and `/root`.
5. Save. The page will be published after GitHub Pages finishes building.
